# Domination
A C implementation of the boardgame 'Domination' using the ncurses library.

### Tools
- C
- ncurses (not available on windows)

### Running the game
1. Clone the repository  
2. Run these commands  
```console
$ cd Domination
$ make
$ ./domination
```

### Controls
Hit tab in the game and a manual will pop up.
